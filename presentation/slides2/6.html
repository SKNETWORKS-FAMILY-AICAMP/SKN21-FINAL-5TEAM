LLM 단독 응답이 아니라,
문맥과 도구를 반영한 서비스형 구조

질문을 바로 LLM에 보내지 않고, 먼저 문맥과 권한을 확인한 뒤 필요한 tool과 데이터를 선택합니다.

01

사용자 질문

위젯에서 상담 요청이 시작됩니다.

02

문맥 · 인증 확인

현재 화면과 로그인 여부를 먼저 확인합니다.

03

Tool 선택

질문 유형에 맞는 공개 / 보호 도구를 고릅니다.

04

데이터 조회

상품, 정책, FAQ, 주문 상태를 필요한 만큼 가져옵니다.

05

SSE 스트리밍 응답

근거를 반영한 응답을 스트리밍으로 반환합니다.
