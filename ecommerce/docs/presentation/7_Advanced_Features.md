# 7. 프로젝트 기능 고도화 방향

단기적으로 완성된 E-Commerce & AI Agent 통합 플랫폼 위에, 차기 스프린트(Sprint)에서 실제 비즈니스 임팩트(매출 전환, MAU 활성화)를 낼 수 있는 스케일업(Scale-up) 로드맵입니다.

## 기능 확장 및 아키텍처 고도화 플랜 (Next Sprint)

1. **상품 검색 부분 초고도화 (Search & Recommendation Enhancements)**:
   - 현재 작동하는 하이브리드(BM25+Dense) 검색을 기반으로 유저의 이전 리뷰 평점, 구매 이력을 가중치로 부여하는 초개인화 엔진(Collaborative Filtering) 도입.
   - 텍스트 오타나 부정확한 질의("예전에 그 원피스 비슷한 거")도 Reranker의 성능 튜닝을 통해 95% 이상 정확도로 추천 상품을 찾아올 수 있도록 파이프라인 고도화 검토.

2. **C2C '유즈드(Used, 중고)' 마켓 카테고리 인프라 편성**:
   - 일반 B2C 쇼핑 한계 돌파를 위한 개인 간 중고물품 거래 기능 신설.
   - 중고 상품 등록 시 멀티모달 LLM 비전 모델을 활용하여 사용자가 업로드한 이미지를 분석, 사용감(A/B/C 등급)을 측정하고 적정 거래가를 제안하는 Pricing Agent 신규 투입 연구.

3. **고가용성 분산 처리 & 캐싱(Caching) 레이어 신설**:
   - 상품 카탈로그 등 변동 빈도가 낮은 데이터에 대해 Redis 캐싱 서버를 백엔드에 추가 부착하여 서버 I/O 부하와 DB 트래픽을 기존 대비 40% 이상 저감하는 백엔드 최적화 추진.
   - Human-in-the-Loop 관리자 경험(UX) 개선의 연장선으로, 장애 시나리오나 에이전트 답변 품질 미달을 감지한 뒤 실시간으로 관리자가 채팅권을 빼앗아오는(Take-over WebSocket 통신) 무중단 CS 채널 추가.
